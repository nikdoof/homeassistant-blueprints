# homeassistant-blueprints

A collection of Home Assistant blueprints for use with various integrations. 

## Zigbee2MQTT
 
 A set of blueprints to be used with [zigbee2mqtt](https://www.zigbee2mqtt.io) and various Zigbee devices. These assume that you're running Zigbee2MQTT v1.22.0 or later, and Home Assistant 2021.12.1 or later.
